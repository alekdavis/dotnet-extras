Converts any object to a dynamic object.

Parameters

source  T
The original object.
extras  DictionaryString, Object  (Optional)
Additional propertyNames to be added to the expando object.

If true, only public properties will be included.

Type Parameters

Data type of the source object.

Usage Note

In Visual Basic and C#, you can call this method as an instance method on any object of type T. When you use instance method syntax to call this method, omit the first parameter. For more information, see Extension Methods (Visual Basic) or Extension Methods (C# Programming Guide).
Remarks
Adapted from https://stackoverflow.com/questions/42836936/convert-class-to-dynamic-and-add-propertyNames#answer-42837044. For dictionary objects, use the ToDynamic(DictionaryString, Object) method instead.
Example
C#
Copy
User user = new()
{
    Name = new()
    {
        GivenName = "John",
        Surname = "Doe"
    },
    Age = 42,
    Mail = "John.Doe@mail.com",
};

Dictionary<string, object> extras = new()
{
    { "ExtraProperty", "XYZ" }
};

dynamic? result = user.ToDynamic(extras);
