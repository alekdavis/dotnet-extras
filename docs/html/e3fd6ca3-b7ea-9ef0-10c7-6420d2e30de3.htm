      DotNetExtras.Common Namespace

      Provides general-purpose utility classes that extend functionality of most frequently used types, such as exceptions and strings.

      Classes

      AssemblyInfo Use the static AssemblyInfo methods and properties to access the primary application assembly and its most commonly used attributes (the primary assembly normally corresponds to the main application assembly).
      Extensions Use the static Extensions methods as shortcuts for common operations on various data types.
      NameOf Use the static NameOf methods to generate fully qualified or partial names of variables, types, or members (you can keep the original case or use the camelCase notation). Think of the NameOf class as the nameof expression on steroids.
      SafeException Use the SafeException class as the base exception for your custom exception classes, so you can easily recognize them in code. This can be handy in a few cases. For example, calling the GetMessagesT(Exception, Boolean) extension method passing SafeException as the generic type (or calling the GetSafeMessages(Exception, Boolean) extension method), will only return messages from your custom exceptions which can help you control the error details sent to the other apps and make sure sensitive information is not leaked outside of your application.
