# DotNetExtras.WebClient Library

The `DotNetExtras.WebClient` library can simplify client code necessary for making REST API calls and processing results. It enables the caller to automatically refresh bearer tokens, translate most common web API errors into appropriate exceptions, and perform other common tasks with fewer lines of code. For HTTP requests, `DotNetExtras.WebClient` utilizes the [Flurl library](https://flurl.dev/) library.

For details, check the topics under the `DotNetExtras.WebClient` namespace section in the [API documentation](https://alekdavis.github.io/dotnet-extras).
