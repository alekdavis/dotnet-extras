# DotNetExtras.Security Library

The `DotNetExtras.Security` library implements general-purpose classes that can simplify common tasks in various .NET Core projects. 

For details, check the topics under the `DotNetExtras.Security` namespace section in the [API documentation](https://alekdavis.github.io/dotnet-extras). For additional usage examples, see the [unit tests project](../SecurityLibTests).
