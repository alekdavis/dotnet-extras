CryptoDecrypt Method

Decrypts cipher text using a 256-bit AES key and a 128-bit initialization vector (IV) derived from the caller-provided password and random 8-character salt.

Remarks
To generate the AES key, the code must use the same password as the one used to encrypt plain text and the 8-character salt included in the beginning of the cipher text that was randomly generated and used for key generation during encryption.
