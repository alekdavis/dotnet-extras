      AssemblyInfo Class

      Use the static AssemblyInfo methods and properties to access the primary application assembly and its most commonly used attributes (the primary assembly normally corresponds to the main application assembly).

      Example

      C#
      string company   = AssemblyInfo.Company;
string copyright = AssemblyInfo.Copyright;
string product   = AssemblyInfo.Product;
string version   = AssemblyInfo.Version;
string title     = AssemblyInfo.Title;

      Properties

      Company Returns the company name.
      Copyright Returns the copyright message.
      Description Returns the assembly description.
      Product Returns the product name.
      Title Returns the assembly title.
      Version Returns the version of the assembly file.

      Methods

      GetAssembly Returns the main application assembly or the assembly implementing the specified type.
