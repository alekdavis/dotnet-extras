Remarks
You can pass array items by appending zero-based index to the key as illustrated in the example (e.g."ServiceA:ArraySetting1:0", "ServiceA:ArraySetting1:1", etc.). See https://stackoverflow.com/questions/37825107/net-core-use-configuration-to-bind-to-options-with-array for details.
Example
IConfiguration config = SetConfiguration( new Dictionary<string,string?> { {"ServiceA:ValueSettingX", "ValueX"}, {"ServiceA:ValueSettingY", "ValueY"}, {"ServiceA:ValueSettingZ", "ValueZ"}, {"ServiceA:ArraySetting1:0", "Value0"}, {"ServiceA:ArraySetting1:1", "Value1"}, {"ServiceA:ArraySetting1:2", "Value2"}, } );
