      DotNetExtras.Security Namespace

      Implements security capabilities, such as encryption, hashing, random password generation, and so on.

      Classes

      Crypto Encrypts and decrypts text using the AES algorithm with 256-bit key, 128-bit initialization vector, 1000 password iterations SHA-256 hash algorithm, and random 64-bit salt.
      Hash Implements hashing functions.
      JsonCharMaskAttribute Applies a character mask to a string property during JSON serialization using `System.Text.Json` (STJ).
      JsonHashMaskAttribute Applies a hash mask to a string property during JSON serialization using `System.Text.Json` (STJ).
      JsonLiteralMaskAttribute Allows applying masks to protect sensitive string object properties.
      Jwt Class for generating and validating JWTs.
      Password Generates random password, which will not include ambiguous characters, such as I, l, and 1. The generated password will be made of 7-bit ASCII symbols. very four characters will include: - one lower case character - one upper case character - one number - one special symbol (such as '%') in a random order. The password will always start with an alpha-numeric character; it will not start with a special symbol (because some back-end systems do not like certain special characters in the first position).

      Enumerations

      HashType Types of hashing algorithms supported by .NET Core.
