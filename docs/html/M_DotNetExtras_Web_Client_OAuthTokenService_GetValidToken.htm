      secondsBeforeExpiration  Int32  (Optional)
      Number of seconds before the calculated expiration when the token needs to be renewed.
      secondsAfterCreation  Int32  (Optional)
      Optional number of seconds after generation when the token must be renewed (when set to zero, which is the default, the setting will be ignored).

      Return Value

      OAuthToken
      Current or renewed access token.

      Remarks

      This function checks the calculated token expiration and if it is within the allowed threshold, it will return the existing token. If the expiration is beyond the threshold, the function will attempt to renew the token. If the token renewal attempt fails, the function will check if the token is past the expiration and if so, it will throw an exception; if not, it will return the existing token, because it may still be good.
