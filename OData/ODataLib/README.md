# DotNetExtras.OData Library

The `DotNetExtras.OData` library implements general-purpose classes that can simplify common tasks in various .NET Core projects. 

For details, check the topics under the `DotNetExtras.OData` namespace section in the [API documentation](https://alekdavis.github.io/dotnet-extras). For additional usage examples, see the [unit tests project](../ODataLibTests). Also check the demo projects that illustrate the usage of the OData library:

- [ODataFilterParserDemo](../ODataFilterParserDemo)
- [ODataFilterValidatorDemo](../ODataFilterValidatorDemo)
