DotNetExtras.Common Namespaces

Contains general-purpose utility classes and specialized namespaces.
Namespaces
NamespaceDescription
DotNetExtras.CommonProvides general-purpose utility classes that extend functionality of most frequently used types, such as exceptions and strings.
DotNetExtras.Common.EnumsProvides utility classes that enhance the capabilities and consumption of the enum types.
DotNetExtras.Common.JsonProvides simplified JSON capabilities, such as conversion of JSON strings to objects and objects to JSON strings.
DotNetExtras.Common.RegularExpressionsDefines regular expressions for common values, such as email addresses, GUIDs, names, and so on.
DotNetExtras.Common.SpecializedProvides more specific and less frequently used general-purpose utility classes that extend functionality of various types.
