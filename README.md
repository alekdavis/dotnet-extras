# DotNetExtras libraries
This repository contain the source code of the DotNetExtras libraries, which are designed to simplify common tasks in .NET Core projects. The libraries are organized into several projects, each serving a specific purpose.

## DotNetExtras library projects

### DotNetExtras.Common library
Implements general-purpose classes that can simplify common tasks in various .NET Core projects.

### DotNetExtras.Mail library
Provides functionality simplifying the use of localized email templates in .NET Core projects.

### DotNetExtras.OData library
Offers functions for parsing and validating OData search filters.

### DotNetExtras.Security library
Simplifies security operations such as hashing, encryption, random password generation, etc.

### DotNetExtras.Testing library
Contains helper classes that can be used in unit tests.

### DotNetExtras.Web library
Defines helper methods for REST API clients.
