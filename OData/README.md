# DotNetExtras.OData library projects
This folder contains the source code of the `DotNetExtras.OData` library and related projects. 

## ODataLib project
Implements the `DotNetExtras.OData` library.

## ODataLibTests project
Implements unit tests for the `DotNetExtras.OData` library.

## ODataFilterParserDemo project
Illustrates the functionality of the OData filter parser implemented in the `DotNetExtras.OData` library.

## ODataFilterValidatorDemo project
Illustrates the functionality of the OData filter validator implemented in the `DotNetExtras.OData` library.

## ODataSampleModels project
Implements sample models used by the `DotNetExtras.OData` library demo and unit test projects.
